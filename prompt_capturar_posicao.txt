Atenção: Antes de qualquer outra ação, verifique a lista de arquivos mais recente na pasta @capturas.

Obter as posicoes gravadas em todos os arquivos de imagem na pasta @capturas.

Gravar o arquivo @bova.csv no formato 'quantidade;ativo'.

Regras:
1. Para cada imagem encontrada na pasta, extrair a quantidade e o código do ativo de cada opção.
2. Incluir no arquivo de saída apenas as opções cujo código do ativo começa com 'BOVA'
3. Se uma mesma opção (mesmo código de ativo) aparecer em mais de uma imagem, incluí-la apenas uma vez no arquivo final.
